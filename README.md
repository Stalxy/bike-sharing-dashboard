# 🚴 Bike Sharing Dashboard ✨

## Setup Environment

### 1. Create Environment
```
bash
conda create -n main-ds python=3.9
```

### 2. Activate Environment

```
conda activate main-ds
```

### 3. Install Dependencies
```
pip install -r requirements.txt
```

## Run Streamlit App
```
streamlit run dashboard.py
```
